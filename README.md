## Description

Boilerplate is [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Clone the repository ###

```bash
$ git clone https://gitlab.webcase.me/Node-projects/boilerplate/nest-boilerplate.git
```

## Installation ###

```bash
$ cp -r .env.dev .env
$ yarn install
```

Update enviroment variable in .env file

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# debug watch mode
$ yarn run start:debug

# production mode
$ yarn run start:prod
```

## Format and es6 checker

```bash
# Format the code using prettier
$ yarn run format

# es6 checker
$ yarn run lint
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

# watch test
$ npm run test:cov

# debug test
$ npm run test:debug
```
